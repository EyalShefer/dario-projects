# Resources

**Reference materials, templates, and skill documentation.**

Use this folder for knowledge assets that support your projects and areas. This is your personal library.

## Categories

- **Templates:** Reusable formats (meeting notes, project briefs, etc.)
- **Guides:** How-to documents, checklists, workflows
- **Reference:** Cheat sheets, API docs, tool notes
- **Skills:** Documentation on tools, languages, frameworks
- **Media:** Images, icons, design assets

## Format

Organize by topic:
```
resources/
  templates/
    project-brief.md
    meeting-notes.md
  guides/
    remote-work-setup.md
    backup-procedures.md
  skills/
    python/
    web-dev/
  media/
    logos/
    screenshots/
```

## Characteristics

- **Reusable:** Can be applied to multiple projects/areas
- **Reference:** Consulted when needed
- **Static:** Generally updated, not actively worked on

Keep it organized so you can find what you need quickly.
