# Projects

**Active projects with defined outcomes and deadlines.**

Use this folder to track initiatives that are time-bound or have a specific completion goal. Each project should have:

- Clear objective or deliverable
- Expected completion date
- Related files, notes, or subtasks

## Format

Create a subfolder or file for each project:
```
projects/
  project-name/
    README.md        (overview, status, deadline)
    notes.md         (progress, blockers, decisions)
    assets/          (files, resources)
```

## Examples

- Website redesign (Deadline: Q1 2026)
- Course completion (Deadline: March 2026)
- Home automation setup (Deadline: February 2026)

Once a project is complete, move it to `/archive`.
